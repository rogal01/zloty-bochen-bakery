Dlaczego baza danych

Database is the difference between a page and a business tool.

For a bakery, content changes constantly: flavors, pickup places, prices, dates, and customer inquiries. A database keeps that information structured, searchable, editable, and reusable across the public site, API, admin panel, and future dashboard.

Prisma + SQLite

The Next.js app uses typed models for locations, flavors, and orders. Good for modern full-stack TypeScript.

FastAPI + SQLite

The Python service exposes the same data as API endpoints. Any frontend could consume it.

WordPress database

The theme stores editable cakes, locations, and private inquiries in the WordPress admin database.
