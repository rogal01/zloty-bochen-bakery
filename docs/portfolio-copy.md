# Tytuł do portfolio

Złoty Bochen - fikcyjna strona piekarni z konfiguratorem tortów 3D

# Krótki opis

Fikcyjna realizacja dla lokalnej piekarni premium. Strona łączy wizerunkową prezentację marki, ofertę tortów i pieczywa, listę punktów odbioru oraz interaktywny konfigurator tortu z podglądem 3D, wyborem kremu, dekoracji sezonowych i napisu na opłatku.

# Dłuższy opis projektu

Złoty Bochen to kompletna koncepcja strony dla lokalnej piekarni, przygotowana tak, aby pokazać zarówno warstwę wizualną, jak i praktyczne funkcje sprzedażowe. Użytkownik może poznać historię marki, przejrzeć ofertę tortów, sprawdzić lokalizacje odbioru i skonfigurować własny tort przed wysłaniem zapytania.

Najważniejszym elementem jest kreator tortu: pozwala wybrać kształt, rozmiar, smak, rodzaj kremu, kolor, owoce sezonowe, dekoracje kremowe oraz napis na opłatku. Podgląd 3D pomaga klientowi szybciej zrozumieć efekt końcowy, a formularz zbiera dane potrzebne cukierni do potwierdzenia zamówienia.

# Skrócona wersja na kartę projektu

Strona fikcyjnej piekarni premium z ofertą, lokalizacjami i konfiguratorem tortów 3D. Projekt pokazuje nowoczesny frontend, pracę z danymi, formularz zapytania oraz warianty wdrożenia dla różnych typów hostingu.

# Technologie

Next.js, TypeScript, React, Tailwind CSS, Three.js, Prisma, SQLite, FastAPI, PHP, WordPress
