Projekt Wersje Dowody

Fikcyjna marka, realny scenariusz sprzedażowy

Portfolio dla lokalnej piekarni, które pokazuje więcej niż ładny ekran.

Złoty Bochen to anonimowy projekt demonstracyjny: pełny Next.js site z API i konfiguracją tortu, statyczna wersja pod GitHub Pages oraz mały wariant PHP pod tani hosting.

Live Next.js demo Static HTML version

Dlaczego tak

Jedna mocna realizacja, trzy praktyczne sposoby pokazania umiejętności.

Dla klientów freelance liczy się zaufanie: czy strona wygląda profesjonalnie, czy rozwiązuje problem biznesowy i czy da się ją wdrożyć w realnych warunkach. Dlatego projekt pokazuje zarówno nowoczesny stack, jak i prostsze warianty dla budżetowych wdrożeń.

Wersje projektu

Każda wersja ma inny sens sprzedażowy.

Flagship Next.js

Pełna aplikacja: React, TypeScript, Tailwind, Prisma, API routes i podgląd tortu 3D.

Kod źródłowy

Static HTML/CSS/JS

Bez build stepów i backendu. Dobre do taniego hostingu, GitHub Pages i szybkich kampanii.

Otwórz wariant statyczny

PHP inquiry form

Mały formularz zapytania o tort, gotowy do pokazania klientowi z klasycznym hostingiem.

Co klient widzi

Projekt opowiada o wartości biznesowej, a nie tylko technologii.

  • Oferta: pieczywo, torty premium, lokalizacje i click & collect.
  • Konwersja: jasne CTA do konfiguratora oraz zapytania o odbiór.
  • Wiarygodność: fikcyjne dane, brak podszywania się pod realną markę.
  • Zakres: od szybkiego statycznego landing page’a po pełną aplikację.
