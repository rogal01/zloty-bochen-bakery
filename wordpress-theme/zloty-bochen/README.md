# Złoty Bochen WordPress Theme

Portfolio WordPress theme for the fictional bakery brand. It shows a client-editable version of the same concept, which is useful for freelance work because many small businesses already understand WordPress.

## What It Shows

- Custom theme structure: `style.css`, `functions.php`, `front-page.php`, `header.php`, `footer.php`.
- Custom post types for cakes, locations, and inquiries.
- A front-page inquiry form that stores submissions as private WordPress posts.
- Fallback demo content so the theme looks good immediately after activation.

## Install Locally

1. Copy `wordpress-theme/zloty-bochen` into `wp-content/themes/zloty-bochen`.
2. Activate **Złoty Bochen Portfolio Theme** in WordPress admin.
3. Add cake and location entries in the admin, or use the built-in fallback demo content.
4. Set a static front page if WordPress does not show the theme front page automatically.
